Note que estes dois programas fazem o mesmo:

```gobstones
program {
  Mover(Leste)
  Mover(Leste)
  Mover(Sul)
}
```

```gobstones
program {
  Mover(Leste)
  Mover(Sul)
  Mover(Leste)
}
```

Moral: **Não existe uma única maneira de resolver um problema!**

A ordem, às vezes, não é tão importante. Avance para o próximo exercício para entender melhor isso.
